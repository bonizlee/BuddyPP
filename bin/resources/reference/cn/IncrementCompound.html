符合运算 += , -= , *= , /=

描述

用另一个常数或变量对变量执行数学运算。+= 运算符只是扩展语法的一种方便速记，如下所示。

语法

x += y;   // equivalent to the expression x = x + y;
x -= y;   // equivalent to the expression x = x - y; 
x *= y;   // equivalent to the expression x = x * y; 
x /= y;   // equivalent to the expression x = x / y;

参数

x: 任意变量类型

y: 任意变量类型或常量

例子

x = 2;
x += 4;      // x now contains 6
x -= 3;      // x now contains 3
x *= 10;     // x now contains 30
x /= 2;      // x now contains 15
