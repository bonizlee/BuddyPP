unsigned long

描述

无符号长整型变量扩充了变量容量以存储更大的数据，它能存储位(4字节)数据。与标准长整型不同无符号长整型无法存储负数，其范围从0到4,294,967,295（2 ^ - 1）。

例子

unsigned long time;

void setup()
{
	Serial.begin(9600);
}

void loop()
{
	Serial.print("Time: ");
	time = millis();
	//prints time since program started
	Serial.println(time);
	// wait a second so as not to send massive amounts of data
	delay(1000);
}

语法

unsigned long var = val;

  • var - 你所定义的变量名
  • val - 给变量所赋的值
